# AI 智囊圆桌 UI 空间感与点击坐标定位指南

本指南记录了本应用各屏核心 UI 控件的物理坐标分布。这能让进入工作区的 AI 代理在进行 UI 截图调试、自动点击以及界面空间布局优化时，获得清晰精确的**“空间感”与“三维尺度定位坐标系”**。

---

## 1. 标准物理坐标系与缩放假设

- **基准分辨率**：所有坐标均基于标准 Android $1080 \times 2400$ 像素布局（由模拟器或高密度竖屏拉取所得）。若在不同物理分辨率设备上运行，可根据比例进行线性缩放。
- **状态栏与虚拟导航栏**：
  - 顶部状态栏区间：Y 为 `0` 至 `80` 左右。
  - 底部系统虚拟导航栏区间：Y 为 `2350` 至 `2400+`。
- **自动化定位技巧**：首选使用 `python tools/uidump.py --find "<文字>"` 本地极速提取控件中点坐标。若文字不固定或为占位符，可使用本指南中提供的绝对/相对位置坐标进行盲点。

## 6. 跨设备等比例缩放与真机映射

为了打通模拟器自动化调试与真机（小米 14 Ultra）的无缝切换，系统梳理了两个设备的等比放缩关系：

### 6.1 设备规格对照
- **基准设备 (模拟器)**：宽度 1080，高度 2400，DPI 420 (屏幕纵横比 20:9)。
- **真机设备 (小米 14 Ultra)**：宽度 1440，高度 3200，物理分辨率 (屏幕纵横比 20:9)。
- **屏幕比例判定**：
  $$2400 / 1080 = 2.222\dots \quad \text{vs} \quad 3200 / 1440 = 2.222\dots$$
  两者的物理比例完全一致，均为 **20:9**。这保证了界面在等比例缩放时没有任何裁切变形，所有触控控件的位置仅发生了纯粹的拉伸。

### 6.2 坐标放缩公式 (放缩因子 $\alpha = \frac{4}{3} \approx 1.33333$)
如果在模拟器上获取到的中点坐标为 \((X_{emu}, Y_{emu})\)，在小米 14 Ultra 真机上对应的物理点击坐标 \((X_{phone}, Y_{phone})\) 为：
\[X_{phone} = \text{round}(X_{emu} \times 1.33333)\]
\[Y_{phone} = \text{round}(Y_{emu} \times 1.33333)\]

反之，从真机映射回模拟器：
\[X_{emu} = \text{round}(X_{phone} \times 0.75)\]
\[Y_{emu} = \text{round}(Y_{phone} \times 0.75)\]

## 7. 无文本图标的语义 & 视觉混合定位

由于部分核心控件（例如菜单、齿轮、加号、星标）不含常规 Text 文本，项目提供了两种高级定位路径，支持 AI 代理完美进行“盲操”：

### 7.1 XML 智能同义词推荐定位 (`uidump.py`)
为了避免盲猜 content-description，在 `uidump.py` 的 `--find` 查询接口中，内置了高频图标别名机制：
* **抽屉 / 菜单**：用户输入“菜单”或“抽屉”，会自动关联并匹配 `menu`, `drawer`, `navigation`, `打开导航抽屉` 等属性。
* **设置 / 齿轮**：用户输入“设置”或“配置”，会匹配 `setting`, `setup`, `config`, `密钥配置` 属性。
* **添加 / 加号**：用户输入“添加”或“加号”，会匹配 `add`, `plus`, `添客`, `录入新席位` 属性。
* **保存 / 星标**：用户输入“存组”或“保存”，会匹配 `save`, `group`, `另存为分组` 属性。

* 使用示例：
  ```bash
  python tools/uidump.py --find "设置"
  # 输出: 975 142
  ```

### 7.2 视觉级多尺度图像模板匹配定位 (`find_icon.py`)
当 XML 树无任何文本反馈（如 Canvas 绘制的极简占位符），或在真机与模拟器 DPI 变化时，可以通过预存的图标特征图进行**多尺度视觉搜寻**：
- **原理**：调用 [find_icon.py](file:///d:/My_Elio/AI-Skill-Roundtable/tools/find_icon.py) 读取 `tools/templates/` 目录下的基准图标（如 `setting.png` 或 `menu.png`），自动截取当前屏幕，并进行 $0.5 \sim 1.5$ 倍的多尺度滑动窗口匹配。
- **预设模板位置**：
  * 齿轮设置特征图：[setting.png](file:///d:/My_Elio/AI-Skill-Roundtable/tools/templates/setting.png)
  * 三横菜单特征图：[menu.png](file:///d:/My_Elio/AI-Skill-Roundtable/tools/templates/menu.png)
- **使用示例**：
  ```bash
  python tools/find_icon.py -t tools/templates/setting.png
  # 自动自适应缩放并输出: 975 142
  ```
- **点击直连**（结合 click.py 完成盲操点击）：
  ```powershell
  # 在 PowerShell 中获取坐标并直接点击：
  $coords = python tools/find_icon.py -t tools/templates/setting.png
  python tools/click.py $coords
  ```
