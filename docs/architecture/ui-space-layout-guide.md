# AI 智囊圆桌 UI 空间感与点击坐标定位指南

本指南记录了本应用各屏核心 UI 控件的物理坐标分布。这能让进入工作区的 AI 代理在进行 UI 截图调试、自动点击以及界面空间布局优化时，获得清晰精确的**“空间感”与“三维尺度定位坐标系”**。

---

## 1. 标准物理坐标系与缩放假设

- **基准分辨率**：所有坐标均基于标准 Android $1080 \times 2400$ 像素布局（由模拟器或高密度竖屏拉取所得）。若在不同物理分辨率设备上运行，可根据比例进行线性缩放。
- **状态栏与虚拟导航栏**：
  - 顶部状态栏区间：Y 为 `0` 至 `80` 左右。
  - 底部系统虚拟导航栏区间：Y 为 `2350` 至 `2400+`。
- **自动化定位技巧**：首选使用 `python tools/uidump.py --find "<文字>"` 本地极速提取控件中点坐标。若文字不固定或为占位符，可使用本指南中提供的绝对/相对位置坐标进行盲点。
