# ADR-007: Notion 极简风 UI 重构与物理阻尼 Spring 交互动效决策

## 1. 状态
**已确认 (Accepted)**

## 2. 背景与动机
在 v2.1 交付后，应用虽然功能完善，但面临显著的视觉与交互痛点：
- **纵向空间极度局促**：顶部的“智囊头像列表”、“联网搜索选择胶囊”以及“自动顺延/余弦排序”两个 Switch 开关占用了将近 260dp 的垂直空间，在标准竖屏手机上吃掉了 30% 以上的阅读视窗，导致聊天气泡极度受限。
- **Slop 廉价感严重**：全屏充斥着粗糙的 Emoji（如 🧠, ⚡, 🔮），卡片边框厚重粗糙，按钮和巨大绿 Switch 满屏。
- **静态僵硬**：点击按钮时，缺乏现代高级 App 灵动的微动效与触觉反馈。

为了提升视觉审美上限（Notion 极简风、Linear 线性风），必须对主屏幕与“智囊大厅（Tab 1）”进行整体的物理重构与空间压缩。

## 3. 决策内容

### 3.1 物理阻力弹簧回弹微交互 (`bounceClick`)
- **实施**：编写一个自定义 `Modifier.bounceClick()`，使用 Jetpack Compose `pointerInput` 手势检测点击态，利用 `Animatable(1f)` 在点击按下时收缩至 `0.96f`，在松开或取消时以 **Spring 物理弹簧弹力** 阻尼平滑回弹。
- **效果**：对应用中所有 TabItem、IconButton、Chip、Card 以及状态胶囊无感挂接，极大地提升了触碰的段落阻尼质感。

### 3.2 去 Emoji 与高定 Canvas 占位符
- **实施**：清除全屏所有带有廉价感的 Emoji (大脑、闪电、魔法球等)，改以纯文本优雅排版。
- **发光脉动环**：设计 `MinimalistPulseIndicator` 代替 🧠 大脑，利用 Canvas 实时计算三层同心圆的 alpha 脉动动画，构成极简的 AI 思考指引。
- **多波条形图**：设计 `MinimalistAudioEmptyIndicator` 代替 🎵，利用 6 根随时间波动的条形矩形构成极简音频声谱波形，用作音频库空状态。

### 3.3 主脑暴屏顶部 Layout 拍扁与开关迁移
- **移除开关**：将原本堆在顶部的“自动顺延发言”和“专家自适应排序”两个 Switch 开关移出主屏，重新收纳在左侧滑 Drawer（会议历史抽屉）的底部，构成偏好设置区。
- **单行横向并排**：名人头像 LazyRow 尺寸由 52dp 降为 32dp，去除汉字文本；联网搜索胶囊（“智能/强制/关闭”）贴紧右侧，与头像列表强行并列为单行 Row。
- **推理微挂件**：TopAppBar 里的“思考中”文案移除，简缩为标题“AI智囊圆桌”旁的一个超窄 12dp 的 `CircularProgressIndicator` 推理指示灯。

### 3.4 智囊大厅 (Tab 1) Bento 线性名册重塑
- **操作隐藏**：废除卡片内部的粗分割线和底部显式的“修改Prompt设定”与“请离会议”两个 TextButton。改为点击卡片本身拉起画像，右侧增加 `MoreVert` 图标，点击后浮现 DropdownMenu 浮动层来托管这些低频高危功能。
- **微型胶囊 Switch**：用圆角 `6.dp`、薄荷绿（入席）和灰色（旁听）的微型“状态胶囊 Box”代替亮绿色大 Switch，极易点击且大幅瘦身。
- **行高压缩**：单张卡片高度由 `140dp` 压缩为 `64dp`，一屏可同时完整陈列 6 个智囊，信息密度和呼吸感极大改善。

## 4. 后果与影响
- **视野释放**：为主屏省下了约 `140dp` 的高度，聊天气泡视窗面积直接扩大了一倍，主屏呈现开阔的 Notion 画册风。
- **高定质感**：界面干净整洁、毫无杂质，触感平滑弹性，达到了极具现代审美的昂贵设计感。
- **兼容性**：不改变任何底层 Character 与 ChatSession 的持久化结构，纯 UI 层无感重构。
