# PR 03：隐私、遥测与数据安全收口

> 优先级：P0
> 覆盖：Prompt 日志、响应日志、本地存储、隐私说明

## 目标
调试能力和用户隐私之间建立明确边界。

## 任务清单

### 1. 遥测分级
默认只保存：
- 模型
- 状态码
- 延迟
- Key 标识
- 错误类型

正文日志必须手动开启。

### 2. 内容脱敏
- 限制长度。
- 脱敏手机号、邮箱、Token。
- 附件禁止默认解析保存。

### 3. 存储治理
- 遥测独立存储。
- 增加 TTL 自动清理。
- 增加一键清空。

### 4. Release 安全
- 禁止 BODY 日志。
- 禁止完整 Prompt 写入 Logcat。

### 5. 云端 Interaction 说明
明确 `store=true` 行为。
增加隐私说明文档。

## 验收标准
- 默认运行不保存完整用户内容。
- 用户可查看、删除调试数据。
- Release APK 无敏感日志。

## 禁止事项
- 不为了方便调试永久保存 Prompt。
